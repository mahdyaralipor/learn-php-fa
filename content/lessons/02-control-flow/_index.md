# فصل دوم: جریان کنترل

تا اینجا یاد گرفتی داده را در متغیر و آرایه نگه داری.
اما برنامهٔ واقعی فقط خط‌به‌خط جلو نمی‌رود — بسته به شرایط تصمیم می‌گیرد و کارها را تکرار می‌کند.

این فصل دقیقاً همان‌جاست: **شرط، انتخاب، حلقه و عبارت‌های کوتاه تصمیم‌گیری**.

## چرا این فصل مهم است؟

بدون جریان کنترل، برنامه‌ات مثل یک دستورالعمل ثابت است که همیشه همان کار را می‌کند.
با شرط و حلقه می‌توانی:

- ورودی کاربر را بررسی کنی
- خطا را مدیریت کنی
- لیست داده را پیمایش کنی
- منطق پیچیده را قابل‌خواندن کنی

## در این فصل چه می‌گیری؟

- `if`, `elseif`, `else` برای تصمیم‌گیری
- `switch` و `match` برای انتخاب بین چند حالت
- `while`, `do-while`, `for`, `foreach` برای تکرار
- عملگر سه‌تایی، null coalescing و nullsafe برای کد فشرده و خوانا

## پیش‌نیاز

این فصل روی مبانی فصل اول سوار است:

- نوع داده و تفاوت `==` با `===`
- عملگرهای منطقی `&&`, `||`, `!`
- آرایه و `foreach`

اگر هنوز با falsy و type juggling راحت نیستی، یک‌بار درس نوع‌ها را مرور کن.

## یک پیش‌نمایش

تا پایان فصل باید بتوانی چنین کدی را بخوانی:

```php
<?php

$role = 'editor';
$permissions = match ($role) {
    'admin' => ['read', 'write', 'delete'],
    'editor' => ['read', 'write'],
    default => ['read'],
};

foreach ($permissions as $permission) {
    echo $permission . PHP_EOL;
}
```

## قانون‌های طلایی فصل

- در شرط‌ها به‌صورت پیش‌فرض `===` بزن.
- شرط‌های پیچیده را به متغیر یا تابع با نام خوانا بسپار.
- حلقهٔ بی‌پایان عمدی نساز مگر بدانی چطور `break` می‌زنی.
- `match` را برای انتخاب مقدار ترجیح بده؛ `switch` را وقتی عادت یا سازگاری لازم است.

## اشتباه رایج

خیلی‌ها فکر می‌کنند اگر برنامه خروجی درست بدهد، شرط‌ها هم درست‌اند.
اما همان `"0" == false` یا `"10" > "2"` می‌تواند منطق شرط را خراب کند.

## تمرین قبل از شروع

این کد را اجرا کن و قبل از خواندن درس بعدی حدس بزن خروجی چیست:

```php
<?php

$score = 85;

if ($score >= 90) {
    $grade = 'A';
} elseif ($score >= 80) {
    $grade = 'B';
} else {
    $grade = 'C';
}

echo $grade . PHP_EOL;
```

## خروجی مورد انتظار

بعد از این فصل باید بتوانی:

- شرط‌های تودرتو و ترکیبی بنویسی
- بین `switch` و `match` انتخاب آگاهانه کنی
- با چهار نوع حلقه کار کنی
- از ternary و `??` برای کد کوتاه‌تر استفاده کنی

از درس بعدی با `if` و `else` شروع می‌کنیم.
