# مبانی وب با PHP

تا اینجا با خود زبان PHP آشنا شدی. از این فصل به بعد وارد بخش وب می‌شیم؛ یعنی جایی که PHP درخواست HTTP می‌گیرد، پردازش می‌کند و پاسخ برمی‌گرداند.

در این فصل چند مهارت پایه را یاد می‌گیری:

1. اجرای سرور داخلی PHP با `php -S`
2. فهم چرخهٔ `request -> PHP -> response`
3. کار با `$_GET` و `$_POST`
4. اتصال فرم HTML به کد PHP
5. آپلود فایل به‌صورت امن
6. ارسال هدر و ریدایرکت

## ذهنیت مهم این فصل

وب یعنی «ورودیِ غیرقابل‌اعتماد». هر چیزی که از URL، فرم، کوکی، هدر، یا فایل آپلود می‌آید باید با شک دیده شود.

از همین فصل این عادت‌ها را پیش‌فرض می‌کنیم:

- برای خواندن ورودی، تا حد امکان از `filter_input()` استفاده کن.
- برای نمایش داده در HTML، از `htmlspecialchars()` استفاده کن.
- برای مقایسه‌های حساس، `===` را به `==` ترجیح بده.
- بعد از ریدایرکت با `header('Location: ...')` همیشه `exit;` بزن.

## خروجی این فصل

وقتی این فصل تمام شود، می‌توانی یک صفحهٔ PHP بسازی که:

- از URL و فرم داده بگیرد
- داده را اعتبارسنجی اولیه کند
- خروجی امن بسازد
- فایل آپلود کند
- در صورت نیاز کاربر را به صفحهٔ دیگر بفرستد

اگر این بخش را خوب بفهمی، فصل‌های امنیت، سشن، و دیتابیس خیلی راحت‌تر می‌شوند.
