# پروژه‌های عملی

درس‌ها مفاهیم را جدا جدا آموزش می‌دهند؛ پروژه‌ها همان مفاهیم را روی یک مسئلهٔ واقعی جمع می‌کنند.
هر پروژه را **بعد از فصل‌های پیش‌نیازش** شروع کن و کد را خودت تایپ کن — copy-paste فقط برای مقایسهٔ نهایی.

## فهرست پروژه‌ها

| # | پروژه | پیش‌نیاز تقریبی | هدف |
|---|--------|-----------------|-----|
| 1 | [ماشین‌حساب CLI](/learn-php-fa/lessons/projects/01-cli-calculator/) | فصل ۱–۳ | تمرین متغیر، شرط، تابع و ورودی خط فرمان |
| 2 | [دفتر نمرات](/learn-php-fa/lessons/projects/02-gradebook/) | فصل ۴ (OOP) | کلاس، property، method و گزارش‌گیری |
| 3 | [TODO با JSON](/learn-php-fa/lessons/projects/03-json-todo/) | فصل ۶–۷ | فرم وب، فایل JSON، اعتبارسنجی و escaping |
| 4 | [ثبت‌نام و ورود با PDO](/learn-php-fa/lessons/projects/04-auth-pdo/) | فصل ۸–۹ | session، hashing، prepared statements |

## ترتیب پیشنهادی

1. **پروژه ۱** — بلافاصله بعد از فصل توابع
2. **پروژه ۲** — بعد از فصل شی‌گرایی
3. **پروژه ۳** — بعد از مبانی وب و امنیت ورودی
4. **پروژه ۴** — بعد از session و PDO
5. **پروژه ۵** — بعد از فصل API
6. **پروژه ۶** — جمع‌بندی نهایی دوره

## روش کار روی هر پروژه

1. هدف و خروجی مورد انتظار را بخوان.
2. ساختار پوشه را خودت بساز.
3. قدم‌به‌قدم پیاده کن — اول نسخهٔ ساده، بعد بهبود.
4. با `php -l` و اجرای دستی تست کن.
5. تمرین‌های انتهای پروژه را انجام بده.

## اگر گیر کردی

- اول با `var_dump` ببین داده چه شکلی است.
- پیام خطا را کامل بخوان، نه فقط آخرین خط.
- نسخهٔ کوچک‌تر بساز که فقط یک قابلیت کار کند.
- درس مربوط به همان مفهوم را یک‌بار مرور کن.

از [پروژه ۱: ماشین‌حساب CLI](/learn-php-fa/lessons/projects/01-cli-calculator/) شروع کن.
