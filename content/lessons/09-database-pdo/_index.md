# دیتابیس با PDO

تا اینجا داده را از کاربر گرفته‌ایم، اعتبارسنجی کرده‌ایم، و در session نگه داشته‌ایم. حالا وقت آن است که داده را پایدار ذخیره کنیم.

در PHP، `PDO` یکی از استانداردترین راه‌ها برای کار با دیتابیس است.

## در این فصل چه یاد می‌گیری؟

- مبانی SQL
- اتصال به دیتابیس با PDO
- prepared statements
- transaction
- ساخت repository ساده

## پیش‌فرض‌های مهم این فصل

از همین اول این سه عادت را قطعی کن:

- `PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION`
- تعیین charset در DSN
- استفاده از prepared statements برای تمام queryهایی که ورودی کاربر دارند

اگر این سه‌تا جا بیفتند، از خیلی از باگ‌ها و خطرها دور می‌مانی.
