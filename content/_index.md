# آموزش PHP از صفر تا پیشرفته

این یه دوره‌ی معمولی PHP نیست. هدف این نیست که فقط سینتکس یاد بگیری — هدف اینه که مثل یه **بک‌اند دولوپر واقعی** فکر کنی: چرا هر مفهوم اونجوری طراحی شده، کجاها اشتباه رایجه، و چطور کدی می‌نویسی که تو پروژهٔ واقعی و تحت لود هم درست کار کنه.

## این مسیر برای کیه؟

کسی که حتی برنامه‌نویسی ندیده، یا فقط مفاهیم پایه رو جایی شنیده، و می‌خواد قدم‌به‌قدم تا سطح حرفه‌ای برسه — OOP، امنیت، دیتابیس، تست، MVC، API و دیپلوی.

## ساختار دوره

۱۵ فصل تو در تو + پروژه‌های عملی در هر مرحله:

| بخش | فصل‌ها | خروجی |
|-----|--------|--------|
| زبان | ۱ تا ۵ | تفکر و کدنویسی تمیز PHP ۸ |
| وب | ۶ تا ۹ | فرم، امنیت، session، PDO |
| حرفه‌ای | ۱۰ تا ۱۵ | Composer، تست، MVC، API، دیپلوی |

هر درس سه بخش داره: **مفهوم** → **کد عملی** → **تمرین**.

## پروژه‌ها

- ماشین‌حساب CLI و دفترچهٔ نمرات
- TODO با JSON
- Auth با Session + PDO
- API JSON
- **اپ MVC نهایی** (بلاگ/تسک‌منیجر)

## چرا PHP؟

PHP هنوز پشت بخش بزرگی از وب (وردپرس، سیستم‌های سازمانی، APIها) قرار داره. تسلط واقعی — نه سطحی — یکی از مطمئن‌ترین مسیرها برای ورود به بک‌اند‌ه.
